MinJs: js文件压缩、合并工具
============================

## Feathures

 * 批量压缩目录下所有js文件
 * 可以通过配置文件排除某些js文件（支持正则）
 * 支持SeaJs模块压缩
 * 开发代码命名`xxx.debug.js`, 发布代码命名`xxx.js`
 * 第一次使用时，若没有`xxx.debug.js`, 会自动创建
